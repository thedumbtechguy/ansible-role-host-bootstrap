# Ansible Role: Host Bootstrap

An ansible role to install required ansible dependencies for hosts.

## Requirements

> This role has been tested on `Ubuntu 16.04` and `Ubuntu 16.10` only.

## Usage Example

```yaml
- hosts: all
  roles:
    - thedumbtechguy.host-bootstrap
```
